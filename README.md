# Reclip Waitlist

The content engine for modern creators. Built in public for those who value their time.

## 🚀 The Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Backend/Storage**: [Make.com](https://www.make.com/) (Webhooks & Google Sheets Integration)
- **Deployment**: Optimized for static hosting

## ✨ Value Propositions
- **10x Distribution**: Turn one video into viral-ready threads, LinkedIn posts, and newsletters in seconds.
- **Efficient Creation**: Reclaim the hours spent on manual content repurposing.
- **Founding Perks**: Early adopters lock in lifetime discounts and exclusive referral rewards.
- **Universal Input**: Whether it's a YouTube link or a clip from your phone, Reclip handles it.
